# Busca-Cep

<h2>:bookmark_tabs: Sobre</h2>
<p>O Busca-CEP é um aplicativo desenvolvido em JavaScript que permite buscar informações sobre endereços a partir de um CEP. Usando API do ViaCep.</p>
